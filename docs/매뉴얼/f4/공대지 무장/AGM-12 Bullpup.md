# AGM-12 Bullpup

## 개요

AGM-12 Bullpup은 현대 정밀유도무기의 “초기형”이라고 볼 수 있는 미사일로 발사 이후 파일럿이 직접 조종 할 수 있다는 특징이 있으나 유도무기의 초기형인만큼 조종성 및 유도 성능은 말 그대로 좋진 않다

장착가능한 불펍은 A,B,C형이 있으며 각 형 별 특징은 다음과 같다

* A형 : 소형 탄두, 소형 표적용, 고체 로켓모터

* B형 : A형과 동일한 소형 탄두, 액체 로켓모터

* C형 : 대형 고폭탄두
