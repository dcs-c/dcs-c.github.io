# 착함

## 개요

호넷은 함재기이기에 항모에서 이륙 할 수 있듯이 반대로 항모에 착함 할 수 있다 공항처럼 한가지 착륙 방법이 정해져 있는것이 아닌 상황에 맞는 착륙 방법이 3종류가 있는데 맑은 낡에는 장주패턴을 돌아 착륙하는 CASE 1, 해질녘이나 우천상황에서 장주패턴 없이 1자로 착함하는 CASE 2, 한밤중 혹은 폭우 등의 가시거리가 전혀보이지 않는 상황에서는 자동착함을 이용하는 CASE 3를 진행한다 심지어 착함이라는게 활주로랑 달리 전투기에겐 정말 좁은 항모 위에 어레스팅와이어를 후크에 걸기 위해 정말 전투기 딱 하나 올라갈만큼의 판자 위로 착륙하는것이기에 공항에서의 착륙보다 체감상 5배는 어렵다 그렇기에 정말 많이 연습해야되고 감을 익혀야 한다
