# 탈드(TALD)

# 개요

탈드는 Tactical Air Launched Decoy, 즉 전술 공중발사 기만체 라는 뜻으로 적의 방공망을 속이는 용도로 쓰는 무장이다
의도적으로 레이더 반사면적을 넓혀 적 지대공 미사일의 레이더에 전투기나 다른 항공기 처럼 보이게 하여 적의 지대공 미사일을 낭비 시키는 목적의 무장이다 거의 내부 시스템은 없다 시피해 그냥 깡통에 날개를 붙여놓은게 끝이라고 할정도로 값싼 무장이기 때문에
적의 고가 지대공 미사일을 빼는데 매우 효과적이다 DCS에서의 지대공 방공망은 서버마다 다르지만 장전하는데 시간이 걸리기에 탈드로 미사일을 빼놓고 무력화 되어있을때 부수는 방법 또한 존재 한다, 사용법 또한 매우 간단하고 한 파일런에 3개씩 장착되어 총 12개의 탈드를 쏠 수 있다
