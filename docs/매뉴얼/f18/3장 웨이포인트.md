# 웨이포인트

## 개요

본 문서는 F/A-18C(통칭 호넷)의 웨이포인트 사용법에 대해 서술하는 문서이다

지상에서 차량에겐 네비게이션이 있고 지도가 있듯이 호넷 및 여타 항공기들에게도 웨이포인트라는 네비게이션이 존재한다

지상 차량은 신호등, 건물, 지형, 다른 차량, 도로 등의 영향을 받아 앞으로 갔다 우회전 좌회전등 길 안내가 복잡한 편이지만 하늘은 장애물이라고는 거의 없기에 웨이포인트는 생각보다 단순한 형태를 띈다 하지만 단순해 보여도 생각보다 간단하진 않으며 활용되는 곳이 많기에
천천히 배울 필요가 있다
