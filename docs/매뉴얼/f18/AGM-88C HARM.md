# AGM-88C HARM

## 개요

기본적인 중거리 이상의 방공체계는 탐색 레이더가 적기를 찾으면 추적레이더가 받은 정보를 토대로 락온을 걸어 기체가 사정권 내로 들어오면 포대에서 지대공 미사일이 발사되는 형식이다

하지만 이를 역이용한 미사일이 대 레이더 미사일, HARM(high speed anti radar missle, 통칭 함)이다 

적 탐색 레이더 및 추적 레이더가 기체를 찾기 위해 내뿜는 레이더파를 역으로 감지해 해당 레이더파의 발원지로 날아가는 미사일이 함이다
