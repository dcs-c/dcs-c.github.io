# 무유도(Unguide) 폭탄

## 개요

기총과 로켓은 타격할 수 있는 범위가 좁은 반면 폭탄은 기총과 로켓에 비해 훨씬 타격범위와 파괴력을 자랑한다 매버릭과 슬램ER같은 미사일 계열의 무장은 추진체나 기타 장치들로 인해 폭약을 많이 넣을 수 없어 파괴력이 낮지만 폭탄은 추진체가 없기에 무장 내부에 훨씬 많은 화약으로 더 강력한 파괴력을 자랑한다 그러한 폭탄중 기본이 되는 무장이 무유도 폭탄이다.

## 사용법

>무유도 폭탄은 여러 종류가 있지만 그중 가장 기본이 되는 MK 시리즈 폭탄을 다룬다

>> MK.82 : 500파운드

>> MK.83 : 1000파운드

>> MK.84 : 2000파운드

>> 무게가 더 무거울 수록 당연히 파괴력이 더 강하다.

> MK 시리즈 폭탄은 2 3 5 7 8번 파일런에 무장이 가능하다

> 호넷은 파일런당 최대 2000파운드의 무장이 가능하기에 2000파운드의 MK.84는 파일런당 1발씩만 장착이 되지만 그 이하인 MK.82,83은 2발씩 무장이 가능하다 

> 2 8번엔 5000파운드 2발, 3 7번엔 1000파운드 2발, 5번에 2000파운드 1발을 무장하겠다


<img width="603" height="503" alt="스크린샷 2025-11-21 103551" src="https://github.com/user-attachments/assets/e2c88d58-2f23-4de6-a418-ec9dd311c9c1" />

> 1. 마스터 암과 A/G모드를 활성화 해준다

> 2. store 화면을 확인해보면 각 파일런에 무장 상태를 확인 할 수 있고 위쪽 버튼에도 3개의 버튼이 활성화 된걸 확인 할 수 있다.
>> - 82B: MK.82
>> - 83B: MK.83
>> - 84: MK.84

<img width="1230" height="1139" alt="스크린샷 2025-11-21 110152" src="https://github.com/user-attachments/assets/99f0d9e8-0b74-473c-8eba-cdb1c300b755" />

> 1. 84 버튼을 눌러 MK.84(2000파운드)를 활성화 해준다

> 2. 활성화가 되면 스토어 화면에 폭탄 관련 설정이 표시 된다

> 3. UFC 버튼, 추후 폭탄 투하량과 거리 등을 입력할때 사용한다

> 4. 기본 설정인 CCIP 모드에 의해 허드에 폭탄의 투하 경로와 투하 지점을 계산해주는 수직 막대가 나타난다

<img width="449" height="443" alt="스크린샷 2025-11-21 110535" src="https://github.com/user-attachments/assets/d80dc688-fed1-4b04-b213-07ecc4d09341" />
<img width="467" height="446" alt="스크린샷 2025-11-21 110638" src="https://github.com/user-attachments/assets/82e09c48-34c5-4cd7-8b65-6978137d6fa2" />

> 1. 좌측 버튼의 MODE를 선택하면 폭탄의 투하 방식을 선택 할 수 있다

> 2. 주로 사용하는 투하 방식으로는 AUTO(CCRP)와 CCIP가 있다
>> - CCIP: 기총과 로켓 처럼 적을 향해 내리 꽂으며 투하 하는 방식으로 호넷 내 컴퓨터가 계산하기 시작해 허드에서 어느 지점에 착탄될지 계산해주는 모드이다
>> - CCRP: 타겟팅 포인트를 지정하면 CCIP처럼 내리 꽂지 않아도 호넷 내 컴퓨터가 해당 타겟팅 포인트를 맞추기 위한 투하 타이밍을 계산해 주는데 때문에 CCIP 모드처럼 다이브 토스를 진행하지 않아도 수평 비행 만으로도 타겟을 맞출 수 있다

## CCIP 투하

> 2. 변경된 모드는 스토어 창 아랫 부분에 표시된다

> 1. MFUZ(기계식 신관) 버튼을 눌러준다 이 버튼은 폭탄의 신관을 설정해주는 모드인데 신관을 활성화 해주지 않고 투하하게 되면 폭탄이 아니라 그냥 엄청 무거운 쇳덩어리를 던진거랑 다름이 없다

> 2. MFUZ 버튼을 누르면 설정할 수 있는 신관 설정들이 표시된다, 본 내용에서는 자주 사용하는 NOSE 신관을 사용하도록 하겠다
>> - OFF: 신관을 끈다
>> - NOSE: 전방신관
>> - TAIL: 후방신관
>> - N/T: 전방신관과 후방신관을 둘다 활성화 한다

> 3. 상태 표시를 통해 신관이 제대로 활성화 되었는지 확인해준다

<img width="554" height="513" alt="스크린샷 2025-11-21 112933" src="https://github.com/user-attachments/assets/fc58be7b-b0c6-48a8-9190-301be1459000" />

> 1. 폭탄 설정이 완료 되고 허드를 보게 되면 FPM 아래로 수직선과 수평선이 교차하는 십자가가 나타나게 될것이다.

>> <img width="948" height="1120" alt="스크린샷 2025-11-21 113646" src="https://github.com/user-attachments/assets/3e309113-be9d-40da-8d2b-ed03d2ea7e71" />

>> - 적을 향해 강하하게 되면 수평선이 점점 수직선의 아래로 이동하게 되고 적당한 투하 각도가 되면 수평선이 수직선 끝으로 이동함과 동시에 수평선이 짧아진다 그러면 그 수평선과 수직선이 이루는 교차점이 탄착지점이다 그 탄착 지점을 적에게 대고 Weapon release 버튼을 누르게 되면 폭탄이 투하 되며 적에게 명중하게 된다

>> <img width="2559" height="1357" alt="image" src="https://github.com/user-attachments/assets/82185d07-29d1-44bc-aa8d-bc77eb70e0ab" />


## CCIP로 투하시 주의 사항

CCIP로 투하시 수평선이 수직선 맨 아래로 내려가서 수평선이 짧아질때 투하해야 원하는 지점에 타격이 가능하며 투하시 고도가 너무 낮거나 혹은 강하각이 완만한 상태로 적에게 접근 하면 수평선이 엄청 천천히 내려온다 때문에 적에게 거의 근접한 상태가 되어야 수평선이 수직선 맨 아래로 내려오게 되는데 이때 폭탄을 투하하게 되면 투하 지점과 기체가 너무 가까워 오히려 임팩시 폭탄의 폭발로 인해 기체가 부숴진다 그러니 최소 7000피트에서 강하 각도 15~20도로 투하하는게 바람직하다  


## CCRP로 투하

CCIP는 적에게 강하를 해야 정확한 투하가 가능했다면 CCRP는 적의 타겟팅 포인트를 찍을 수만 있다면 굳이 기수를 내리지 않고도 어느정도 정확한 타격이 가능하다

>> - MULT: 한번에 투하 될 폭탄의 갯수
