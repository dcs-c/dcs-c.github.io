# 레이저유도폭탄

## 개요

앞서 서술한 월아이는 타겟팅포드가 적을 포착해 주어도 월아이 자체 카메라가 적의 이미지를 포착하지 못하면 사용이 불가하고 그에 따른 오차율도 큰 편이였다 하지만 현 서술할 레이저유도폭탄 Paveway는 타겟팅포드에서 쏴주는 레이저를 지속적으로 쫓아가 움직이는 적 또한 맞출수 있는 유도 폭탄이다
자신의 기체에서 쏴주는 레이저 외에도 아군이 쏴주는 레이저로도 유도가 가능하며 반대로 내가 쏴주는 레이저로 아군이 레이저유도폭탄을 떨궈줄수도 있다 

말만 들으면 가장 좋은 폭탄 같으나 문제는 타겟팅포드로 포착해야 지속적인 레이징이 가능한데 만약 투하 후 크게 기동을 하게되면 타겟팅포드 카메라의 가동범위를 넘어가게되어 타겟을 포착하지 못해 유도가 풀려버려 엉뚱한곳에 떨어지는 일이 발생하기 때문에 적에게 명중하기 전까지는 작전공역을 이탈하기 힘들다는 단점이 있다.

Paveway는 크게 2세대(GBU-10,12,16)와 3세대(GBU-24)가 있다 3세대의 GBU-24 Paveway III가 2세대보다 더 긴 사거리에 정확도로 유도가 가능하며 2세대든 3세대든 기본적인 설정은 같기에 본 문서에선 자주 사용하는 무장인 3세대 레이저유도폭탄 GBU-24 Paveway III로 설명하겠다

> 1. 투하 모드 설정이다.
>> CLAR PP: 진입 사전 방위각을 입력을 하면 그에 맞춰 투하 타이밍을 알려주는 모드다, 오차범위 +- 20도까지 계산해주기에 진입방위각에 얼추 맞춰 입력해줘도된다

>> CLAR SL: PP모드와 달리 현재 진입중인 방위각을 토대로 투하타이밍을 알려준다 보통 이 모드를 선호한다

> 2. 전기신관 설정이다.
>> 작전 상황에 맞춰 설정해주면된다 타겟 타격은 INST, 벙커버스터로는 DLY

> 3. 폭탄이 어떤 레이저 코드를 따라갈지 설정하는 버튼이다
>> 코드가 겹치는 기체들끼리 서로 다른 목표를 레이징하게 되면 본인이 원하는곳이 아닌 엉뚱한곳으로 떨어질 수 있기에 서로 다른 코드를 사용하여 혼선이 없도록 한다

> 4. 어떤 파일런에 있는 폭탄을 사용할지 결정하는 순환 버튼이다

> 5. 기존 무유도 폭탄과 마찬가지로 연속 투하 설정이 가능한 입력 버튼이다
