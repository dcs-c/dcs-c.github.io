# AGM-84D Harpoon

## 개요

본 문서에서 다루는 기체는 호넷, 함재기다 주로 해상 위에서 사용하기에 주 목표는 장갑차나 건물 뿐만이 아닌 선박인 경우도 허다하다 그런 경우에 사용하는 미사일이 AGM-84D Harpoon(통칭 하푼)이다 사거리 약 100마일의 장거리 순항 미사일로 자체 레이더를 통해 방향만 기체가 잡아 날려주면 날아가는 동안에 레이더에 잡히는 적 함선을 타격한다

적 해상 방공을 피하기 위해 수면으로부터 약 50피트 정도의 고도로 저공 비행하며 적 함선의 하단을 공격하여 침몰을 유도하는 skimming 모드와 적 함선의 상단을 공격해 방공을 무력화 시키는 pop up 모드 2가지의 타격방식이 있다
