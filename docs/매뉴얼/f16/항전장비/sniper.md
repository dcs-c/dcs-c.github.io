# AN/AAQ-33 Sniper Advanced Targeting Pod

## AN/AAQ-33 Advanced Targeting Pod
Sniper Advanced Targeting Pod는 F-16C가 목표를 탐지하고 레이저 유도 무장을 유도하기 위한 광학 장비이다. 공대지 임무를 수행하려고 한다면 필수적으로 달아주도록 하자.



2025-12-04일 기준 사용법이 라이트닝 포드보다 복잡하고 열화상 이미지의 화질도 더 구리기 때문에 라이트닝 포드의 이점이 훨씬 커서 잘 사용되지 않는다. 다만 라이트닝 포드는 공식적으로 너프(고증화) 가 예정되어 있는 관계로 미리 익숙해지는 것을 추천한다.



### 라이트닝 포드에 비해 좋은 점
- 훨씬 높은 배율
- 멀티트레킹을 포함한 다양한 기능
- 훨씬 낮은 항력
### 라이트닝 포드에 비해 안좋은 점
- 복잡한 조작
- 특정 조건이 만족되지 않으면 잡히지 않는 초점
- 라이트닝 포드에 비해 낮은 화질(너프 전 기준으로 너프 후에는 라이트닝 포드의 화질이 훨씬 좋다)


## 주요 기능

- 전자광학/적외선(TV/IR)으로 장거리 표적 탐지
- 목표한 지점의 즉각적인 좌표 추출
- 레이저 조사
- 레이저 탐색 및 외부 레이저 추적
- 움직이는 대상 트레킹
- 멀티트레킹

## 장착

동체 우측 하드포인트에만 장착 가능하다. 좌측 동체 하드포인트는 HARM POD만 달린다. 공대공에서도 나름의 쓸모가 있지만 항력과 무게가 증가하니 잘 따져보고 장착하도록 하자.

## 기체 준비 절차
1. 좌측 스틱 우측 제일 앞쪽에 나란히 있는 버튼 4개가 있다. 그 중 왼쪽에서 2번째 버튼이 바로 우측 하드포인트에 전원을 공급하는 버튼이다. 해당 버튼을 올려서 TGP에 전원을 공급해주자.
(TGP 전원공급버튼 사진 삽입)
2. 전원을 켜고 나면 MFD에 TGP 옵션이 생긴다. MFD 아래쪽 버튼에서 사용 할 일이 없는 항목을 두번 누르고 TGP를 선택해 해당 자리를 TGP로 변경해주자. 보통 A/G모드를 선택하면 이미 지정되어 있을 것이다.
3. 이제 해당 버튼을 누르거나 DMS로 화면을 순환하면 TGP화면이 뜰 것이다. 3분 가량의 준비시간이 필요하니 작전 시작할때 미리 켜 놓는걸 추천한다.

## TGP 화면 구성
- FOV : Wide / Medium / Narrow 시야각
- ZOOM : 확대
- AREA / POINT : 추적 모드
- SLAVE : SPI 위치로 센서 정렬
- LSS : Laser Spot Search
- CNTL : 세부 설정
- BORE : 정면 초기화
(TGP 화면과 화면에 뜨는 옵션들을 레이블한 사진 삽입)


## HOTAS 핵심 조작
### SOI 설정
- DMS Down → 해당 MFD를 SOI로 설정



### TDC Slew
- 조준점 이동



### TMS 조작
- TMS Up Short : 트래킹 시도(AREA → POINT)
- TMS Up Long : SPI 생성
- TMS Down : 추적 해제(INR 복귀)
- TMS Right : 추적 모드 강제 전환
- TMS Left : TV, WHITE HOT, BLACK HOT 전환

### RNG knob
- 천천히 확대, 축소

### FOV(핑키 스위치)
-FOV 전환
### 레이저
- Gun trigger 1단계(First Detent): 레이저 수동 조사

### LSS / LST 기능
LSS (Laser Spot Search)

외부에서 비춘 레이저를 탐색

JTAC 협업 시 사용

LST (Laser Spot Track)

이미 감지된 레이저 점을 지속 추적

주의사항

레이저 코드가 맞아야 탐지됨

연기·구름 뒤는 탐지 불가

9. SPI 생성 및 관리
SPI란?

모든 센서 및 무장이 공유하는 “기준 표적점”.

생성

TGP SOI 상태에서 TMS Up Long → SPI 지정

용도

CCRP 투하 기준

다른 센서(HSD, HUD, 웨폰 페이지)와 표적 공유

웨이포인트 슬레이브 가능

10. LGB(GBU-12/10/24) 유도 절차

TGP로 목표 탐색

AREA 또는 POINT Lock

TMS Up Long → SPI 설정

무장 페이지 → GBU 선택

CCRP 모드 설정

릴리즈 큐를 따라 릴리즈 버튼 유지

폭탄 자동 투하

투하 후 8초 뒤부터 레이저 조사(LASE)

표적 명중 확인

안전 회피

11. 스나이퍼 포드 제한사항

Masking(기체에 의해 시야 가림) 발생 시 레이저/IR 차단

INR 모드는 기체 회전 중 일시적 안정화

구름/연기 → IR 감도 저하

지나친 확대 시 흔들림 증가

12. 오류 해결
화면이 검게 변함

냉각 또는 초기화 필요

CNTL → RSET

SOI 적용 안 됨

DMS Down Long 확인

TGP 페이지 위치 확인

레이저 작동 안 됨

L ARM OFF

LTD/R OFF

코드 불일치

방향 이상

BORE 실행

13. 실전 운용 팁

AREA로 먼저 스캔 → POINT로 교전

Markpoint 저장으로 반복 공격 효율 증가

고도와 각도 확보가 유도 성공의 핵심

JTAC 레이저와 코드 반드시 일치 확인

차량은 POINT, 건물은 AREA가 가장 안정적
