# 데이터링크
TNDL(Link-16) WIP

## 심볼
* 적기는 삼각형으로 표시된다.
* 적기인지 아닌지 확인되지 않았다면, 사각형으로 표시된다.
* 아군기는 원형으로 표시된다.

* **전투기 레이더**가 표적을 탐지했다면 속이 꽉 찬 심볼로 표시된다.
* **전투기 레이더**가 표적을 탐지하지 못 했다면 속이 빈 심볼로 표시된다.
* **전투기 레이더**는 내 레이더만을 의미하지 않는다. **편대원이 락온한 적** 역시 전투기 레이더로 취급된다.

## Declutter
기본적으로 F-16의 FCR-데이터링크 심볼은 내가 탐지한 타겟과 아군이 탐지한 타겟을 구분해 보여주지 않는다. 내 레이더가 탐지한 타겟이 무엇인지 확실히 알기 위해서는 Transmit Switch를 통해 Declutter 옵션을 활성화 해야 한다.

기본적으로 **ALL** 필터가 활성화 되어 있다. Transsmit Switch IFF IN 으로 ALL, FTR+, TGTS, NONE 필터를 순환할 수 있다. Transmit Switch IFF OUT 을 누르면 가장 자주 사용하게 될 필터인 FTR+ 를 빠르게 활성화 할 수 있다. 각 필터의 기능은 아래 표를 참조하자.

|   | 데이터링크 심볼 | 아군이 탐지한 타겟 | 아군의 위치 정보 | 내가 탐지한 타겟 |
| - | --- | --- | --- | --- |
| ALL | O | O | O | O |
| FTR+ | O | X | O | O |
| TGTS | O | X | X | O |
| NONE | X | X | X | O |

## Primary Datalink Track
JHMCS를 장착했다면, 다음 절차에 따라 데이터링크로 받아온 타겟 하나의 위치를 HUD와 JHMCS에 표시할 수 있다.

* HSD를 SOI한 다음 데이터링크 타겟 위에 커서를 올린다.
* 데이터링크 타겟 위에서 TMS UP 키를 누른다.
* 다른 타겟을 선택하고 싶다면 TMS RIGHT 키를 누른다.

## Waypoint 전송
Transmit Switch IFF OUT 을 길게 누르면 SPI의 좌표를 TNDL 데이터링크로 전송할 수 있다.

누군가 TNDL로 웨이포인트를 공유했다면, "Data" 알림음과 함께 전송받은 좌표가 웨이포인트 500번부터 순차적으로 저장된다. HUD 에 표시된 정보는 ICP의 WARN REST 스위치를 이용해 제거할 수 있다.
