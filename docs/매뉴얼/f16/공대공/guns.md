# M61 A1 Vulcan Cannon
20mm 리볼버 기관포. 511발의 20mm 기관포탄을 분당 6000발, 초당 100발로 연사할 수 있다. 유럽제 마우저 23mm 기관포나 러시아의 GSh-30 기관포보다는 위력이 부족하지만, 매우 빠른 발사 속도와 넉넉한 장탄수로 명중률을 크게 높였다.

## 종류

| 종류 | 관통력 | 폭발력 | 예광효과 | 용도 |
| - | - | - | - | - |
| AP | 높음 | 없음 | X | 기갑장비 타격 |
| HEI-T | 없음 | 높음 | O | 공대공 예광탄 |
| SAPI-HEI | 보통 | 중간 | X | 다목적 |
