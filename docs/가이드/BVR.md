# 비붕's Guied for BVR

## BVR 이란?
BVR은 Beyond Visual Range, 즉 가시거리 밖 교전을 지칭하는 말이며, 주로 중, 장거리 미사일을 레이더를 이용해 발사하는 방식의 교전이다. BVR의 반대는 WVR(Within Visual Range)로, 가시거리 내에서 교전하는 것을 지칭한다.

에너지의 유지 관리가 핵심이며, 일반적인 경우, '운' 이라는 요소 없이 오직 계산과 예측의 영역에서 승패가 갈린다.

그러나 파일럿이 어느정도 숙련되었을 경우, 서로 미사일 에너지를 잘 계산해서 장거리 저격은 절대 맞아주지 않기 때문에 BVR로는 승부가 나지 않고, 결국 근접에서 WVR로 끝나는 경우가 대부분이다. 


실제로 격주로 열리는 DCSC 서버의 BVR 이벤트에서, 교전 양상이 초반에는 고고도 저격질이 대부분이었지만, 유저들이 고이기 시작하며 고고도는 견제 할 목적으로만 제한된 인원이 올라가고 나머지는 저고도에서 노는 경우가 대부분이다. 어차피 고고도에서 장거리 사격을 하면 상대가 회피하고 미사일 낭비될게 뻔하니 저고도에서 확실한 킬각을 노리겠다는 것. 


다만 고고도에서 견제할 인원조차 없다면 저고도 기체들은 압도적인 에너지 차이로 아무것도 못하고 죽을 확률이 높다. 그러므로 고고도를 완전히 포기하는 것은 그리 좋은 생각이 아니다. 누군가는 올라가서 맞지 않을 것을 알더라도 미사일을 날려 견제를 해야 한다.
