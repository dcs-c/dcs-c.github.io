    웨이포인트¶

    개요¶

    지상에서 차량에겐 네비게이션이 있고 지도가 있듯이 호넷 및 여타 항공기들에게도 웨이포인트라는 네비게이션이 존재한다

    지상 차량은 신호등, 건물, 지형, 다른 차량, 도로 등의 영향을 받아 앞으로 갔다 우회전 좌회전등 길 안내가 복잡한 편이지만 하늘은 장애물이라고는 거의 없기에 웨이포인트는 생각보다 단순한 형태를 띈다 하지만 단순해 보여도 생각보다 간단하진 않으며 활용되는 곳이 많기에 천천히 배울 필요가 있다.

    사용법¶

    1. MFD 홈버튼을 눌러 HSI가 있는 SUPT 페이지로 이동한다.

    2. SUPT 페이지에 HSI 글자가 적혀있는 버튼을 누른다.

    3. HSI를 누르면 해당 페이지가 나온다.

    HSI 내에서는 본 기체를 위에서 바라본 평면으로 방위각, 위치 정보들을 나타내준다.

    1. TACAN: Tactical Air Navigation, 타칸이라고 부르며 각 항모 혹은 공중급유기 등 지정한 타칸 값이 있는데 원하는 항모 혹은 공중급유기의 타칸 값을 입력하면 해당 기체 및 항모가 발산하는 신호를 잡아 HSI에 위치를 표시해준다, 기능은 웨이포인트랑 비슷해보이지만 다른 점이라고 하면 웨이포인트는 좌표를 기준으로 찍어주기에 움직이는 대상을 추적할 수 없지만 타칸은 움직이는 대상에서 나오는 타칸 신호를 계속 갱신 할 수 있어 움직이는 아군 항모의 위치를 정확하게 찾아가는게 가능하다, 물론 각 맵 혹은 서버 마다 타칸 값은 다르기에 브리핑 및 니보드(기본설정 기준 K)를 참고하면 된다. 정확한 사용방법은 착함 문서에서 서술한다

    2. SCL: 쉽게 말하면 HSI가 표시하는 거리다 사진 기준으로 40, 원 중앙에서 원 둘레까지의 거리를 40마일로 표기 하고 그에따라 정보들을 나타내준다. 5/10/20/40/80/160 마일까지 변경이 가능하다, 하지만 너무 멀리 설정해놓으면 10마일 떨어져 있는 상대도 내 기체에 딱 붙어있는것 처럼 표시되기에 적절히 조절해서 사용하는게 좋다.

    3. DATA: 웨이포인트의 정보 값을 입력 및 변경할때 사용한다.

    4. WYPT: 정해놓은 웨이포인트에 대한 거리 및 방위각을 HUD에 표시해준다

    WYPT를 체크 할 경우 네모 박스가 생기며 다음과 같은 정보가 허드에 표시된다.

    • A. 웨이포인트를 향하기 위한 방위각,
    • B. 웨이포인트까지의 남은 거리,
    • C. 웨이포인트 번호

    1. 웨이포인트 번호를 변경 할 수 있다 예를 들어 1번 웨이포인트와 2번 웨이포인트의 정보값을 DATA를 통해 입력후 1번 웨이포인트로 이동한 다음 혹은 필요할때 위쪽 화살표 있는 MFD 버튼(우측 위에 2번째 버튼)을 누를 경우 2번 웨이포인트로 변경된다.

    2. WPDSG: 쉽게 말하면 타겟팅 포인트다, WYPT 버튼이 방위각과 거리만을 표시해준다면 해당 버튼을 누를 경우 허드 및 헬멧 디스플레이에 웨이포인트의 위치가 정확하게 표시된다.

      - 1. HUD에 표시된 다이아몬드 표시가 웨이포인트의 정확한 위치를 표시해준다.

    헬멧 디스플레이는 다음 사진의 노브를 통해 활성화 할 수 있다.

    1. 내 기체를 표시 한것이다 십자가 모양으로 수직 막대가 몸체 수평막대가 좌우 날개다 수직 막대를 수평막대로 나눈것을 기준으로 짧은쪽이 머리 긴쪽이 꼬리방향이다

    2. HSI 상에 선택된 웨이포인트의 위치를 표시

    웨이포인트 입력 방법¶

    웨이포인트는 위치 정보만 있다면 언제 어디서든 입력이 가능하다, 미션 브리핑을 통해 타겟 좌표를 받아 입력 할 수도 있고 혹은 F10을 누르면 전체적인 지도가 나오는데 지도에 우측 ALT + 좌클릭으로 좌표를 딸 수 있다

    우측 ALT+좌클릭을 누를 경우 해당 좌표를 딴 정보가 나오는데 여기서 우리가 사용할것은 "도분초"다

    다시 HSI로 돌아오고 DATA 버튼을 누르면 해당 화면이 뜨는데

    1. 웨이포인트 번호
    2. 웨이포인트에 입력된 좌표 값

    여기서 좌표값을 변경하기 위해 3. UFC 버튼을 누른다

    UFC 버튼을 누르게되면 중앙 키패드(UFC)에 해당 글자들이 뜨게 된다

    1. 좌표
    2. 고도

    1번 POSN의 왼쪽 검은 동그라미 버튼을 누르면 좌표로 선택되고 아까 추출한 좌표값을 N(북위)을 우선 입력후 총 6자리를 순서대로 입력해주고 UFC의 ENTER를 누르면 저장된다 물론 E(동경)도 똑같이 입력해주면된다, 추가로 좌표 입력전 N(UFC 넘버패드 숫자 2), E(UFC 넘버 패드 숫자6)을 우선 입력해야 북위 및 동경 숫자가 얻갈리지 않는다

    좌표 입력 후 고도를 입력해주면 되는데 UFC에서 ELEV를 누른 후 FEET 선택, 아까 추출한 좌표값에서 고도를 입력해주면된다

    그러면 DATA 화면에 좌표값이 변경 되어있는걸 확인 할 수 있다

    웨이포인트 상세 입력 방법¶

    대충 방향을 맞추거나 그 인근에 적을 찾을때 등 대략적인 위치만 알면 찾을 수 있을때 외에 정밀하게 적의 위치를 찾아야 할때가 있다(추후 서술할 공대지 무장 중 폭탄에 해당하는 INS/GPS 유도 무장 JDAM, JSOW와 장거리 순항미사일 SLAM ER등), 그때 "도분초"보다 조금더 상세한 "도분초 상세"를 사용한다. 도분초는 6자지만 도분초 상세는 8자이기에 더 상세한 좌표 설정이 가능하다

    DATA 화면에서 하단 좌측에서 2번째 버튼을 누르면(presice) 조금 더 정밀하게 좌표를 입력 할 수 있다

    좌표 값 입력하는 방법이 조금 달라진다 숫자 입력 창의 공간 한계로 4자씩 끊어 입력해야한다, 앞 4자 입력후 Enter 그리고 다시 뒤에 4자리를 입력해서 북위 8자리 동경 8자리씩 입력하면된다

    변경된 값을 확인한다.

    DATA에서 웨이포인트 좌표 값을 변경후 다시 HSI 화면으로 돌아가고자 하면 기존 DATA 버튼이 있던 MFD 위쪽 버튼 다섯개중 우측 끝에 있는 HSI 버튼을 눌러주면 된다.
